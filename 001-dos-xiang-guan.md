#### 一、Dos 基本命令

**1、打开Dos的命令终端**
```
//1、win+r 弹出功能框

//2、cmd 回车,打开Dos终端
```


<br>
**2、通过Dos 命令设置windows电脑的关机**
- `shutdown /s /t 1800` 设置电脑在1800秒钟后自动关机
- `shutdown /a` 取消已经设置的关机任务


<br>
**3、获取当前, 系统的时间**

```
time
```

<br>
**4、Dos 命令终端清屏**
```
cls
```

<br>
**5、取消当前还未完成的Dos 任务**
```
ctr+c
```

<br>
**6、查看当前电脑的网卡配置信息**

```
ipconfig -all
// 在linux 中是 ifconfig
```

<br>
**07、 检查网络的连接状态**
```
ping www.baidu.com
// 或
ping ip地址
```

#### 二、Dos 文件操作命令

**1、切换盘符**
```
盘符:
比如: d: 切换到D盘
```
只有windows 中才有盘符的概念, MacOs 和Linux 中是不存在盘符的概念的

<br>
**2、切换路径**
在win中只能在当前盘符下切换路径, 是不允许跨盘符切换路径的, 如果要从当前盘符切换到另外的盘符下的路劲,要分步骤进行, 先切盘符再切换路径
```
cd xxx\yyyy // 切换到指定的目录
cd .. // 切换到上一级目录
cd / //切换到当前盘符的根目录
```

<br>
**3、新建目录**
```
md 新目录名 //md 是make directory 的缩写, 和Linux 中的mkdir 功能一样
```


<br>
**4、删除目录**
```
rd 目录 // 删除指定的空目录
rd /s 目录 // 强制删除分控目录
```


<br>
**5、新建文件**
在Dos 中是没有直接创建文件的命令, 只有间接的
```
echo 文件内容>文件名 // 借助重定向 > 创建文件
```

<br>
**6、删除文件**
```
del 被删除文件名
del 目录 // 询问是否删除当前目录下的所有文件
```

<br>
**7、复制文件**
```
copy 路径
```

<br>
**8、移动文件**
```
move 路径 // 必须进入到被移动的文件下
```
